# KarkiAayog

KarkiAayog is a web application built using Next.js and Tailwind CSS to present the **Karki Commission Report** in a structured, searchable, and user-friendly format. It supports bilingual access (English and Nepali) with smart search capabilities, allowing users to easily navigate and explore large document content. The platform focuses on improving accessibility and readability of important public documents through modern web technologies.

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* next-intl (i18n)

## Features

* Bilingual support (English & Nepali)
* Smart search (English → Nepali, Nepali → Nepali)
* Structured document rendering (headings, paragraphs, lists, tables)
* Fast and responsive UI
